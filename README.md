Eat! — лендинг-сайт для приложения доставки еды с кешбэком
Одностраничный сайт для продвижения мобильного приложения «Eat!».

Ключевые особенности
Адаптивный дизайн: вёрстка на HTML/CSS (Flexbox/Grid), 4 брейкпоинта (1024px, 768px, 584px, 480px), бургер‑меню для мобильных.

Интеграция с БД: Supabase (PostgreSQL), таблицы feedbacks (отзывы) и contact_requests (заявки).

Обратная связь: форма с валидацией, дублирование заявок в Supabase и на email (через Web3Forms).

Интерактивность и анимации: плавное появление контента при скролле, каскадное появление элементов, эффект «печатающегося текста» для отзывов, маска ввода телефона, модальные окна.

Дизайн: современный минимализм, модульная сетка (Figma), логотип с фирменным стилем, мокапы приложения.

Технологии
Фронтенд: HTML, CSS, нативный JavaScript.

Бэкенд/БД: Supabase (PostgreSQL).

Сервисы: Web3Forms (email), Vercel (хостинг), GitHub (контроль версий).

Деплой
Автодеплой из GitHub‑репозитория на Vercel.

Ссылка: https://eat-omega-ivory.vercel.app
